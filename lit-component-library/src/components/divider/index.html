Props

Property Type Default Description
dashed Boolean false When true, the divider will be dashed.
Ex:<pl-divider dashed></pl-divider>
orientation String left, right or center The margin-left/right between the title and its closest border is set to '0', when used with orientation, which must be left or right.
Ex: orientationMargin="left"
orientationMargin String left or right The orientationMargin sets the margin of the left or right aligned text to '0'
Ex: orientation="left"
plain Boolean false Divider text show as plain style (not bold)
styles String left or right Allows the user to place inline CSS styles to the 'span' of the divider's text.
type String vertical The direction type of divider. By default the type is horizontal, but when set to vertical, the divider(s) will be vertical.

Using Component in other JavaScript Frameworks

Integrating a Lit Element component with other JavaScript frameworks like Vue, React, and Angular involves creating a wrapper component in each framework to encapsulate and use the Lit Element component. Here are examples for each framework:

Vue.js

In a Vue component, you'd use Divider just like any other HTML element: 

        
          <template>
            <div>
             <pl-divider dashed></pl-divider>
            </div>
          </template>
          
          <script>
            import 'path_to_your_libraries/divider.js'; // Import your Divider component
            
            export default {
                // Your Vue component definition
            };
          </script>
React

In a React component, you'd similarly treat Divider as a regular HTML element: 

        
            import React from 'react';
            import 'path_to_your_libraries/divider.js'; // Import your Divider component
            
            function App() {
              return (
                <div>
                    <pl-divider dashed></pl-divider>
                </div>
              );
            }
            
            export default App;
Angular

In Angular, you'd use Divider as part of your Angular template:

        
            // Import the required decorators and other necessary modules

            @Component({
              selector: 'app-root',
              template: `
                <div>
                    <pl-divider [dashed]="true"></pl-divider>
                </div>
              `
            })
            export class AppComponent {
              // Your Angular component logic
            }

These examples assume that you've configured your framework project to work with web components and that you've imported the Lit Element component correctly. Adjust import paths accordingly.

The interoperability is possible because Web Components, once defined and registered as custom elements, act like native HTML elements that these frameworks can understand and interact with.
