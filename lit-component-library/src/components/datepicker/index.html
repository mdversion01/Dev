Props
Property Type Default Description
append Boolean true This adds an append area to the input field. For these components it contains the calendar icon and for the Date Picker and the Date Range Picker, they are buttons. The default is true.
appendId String This is the id of the append area. When used, the id is added to the append area.
ariaLabel String This is the aria-label for the div that the displays the calendar or calendars. When used, the aria-label is added to the div.
calendar Boolean false When added to the Date Picker component, this will display the stand alone date picker calendar. The default is false. Used only with Date Picker component.
dateFormat String YYYY-MM-DD This is the format of the date. The default is YYYY-MM-DD. It can be set to MM-DD-YYYY instead of the default. The format is based on the moment.js library.
disabled Boolean false When this is true, the input field is disabled and users cannot interact with it.
displayContextExamples Boolean false This is used to display the context examples for the Date Picker component. The default is false.
formLayout String horizontal, inline This changes the layout of the input field. The default is vertical. If input component is used inside a form component, then formLayout prop of form component will be applied to input component and therefore it does not have to be added to the input component. Also, remember to import your components in the correct order.
inputId String This is the id of the input field. When used, the id is added to the input and to the label's 'for' attribute.
is24HourFormat Boolean true This is used with the Date Range Time Picker component. When true, the time is displayed in 24 hour format, when false the time is displayed in 12 hour format. The default is true.
joinBy String - This is the text that is displayed between the start and end dates in the Date Range and Date Range Time Picker components. The default is a hyphen, '-' but you can change the prop to 'joinBy="to"', which is the other acceptable format . This is used only with the Date Range and Date Range Time Picker component.
label String This is the label for the input field. When used the label is added to the aria-label and placeholder (unless placeholder prop is used) attribute.
labelHidden Boolean false This hides the label to be used by screen readers and uses it as the placeholder text (unless placeholder prop is used).
placeholder String This is the placeholder text for the input field. When used, the placeholder is added to the input field. By default, this uses what has been set in the dateFormat prop.
plumage Boolean false This changes the Plumage styles theme. The default is false.
prepend Boolean false This adds a prepend area to the input field. For these components it contains the calendar icon and for the Date Picker and the Date Range Picker, they are buttons. The default is false.
prependId String This is the id of the prepend area. When used, the id is added to the prepend area.
rangePicker Boolean false When added to the Date Range Picker component, this will display the stand alone date range picker calendars. The default is false. Used only with Date Range Picker component.
rangeTimePicker Boolean false When added to the Date Range Time Picker component, this will display the stand alone date range and time picker calendars. The default is false. Used only with Date Range Time Picker component.
required Boolean false This makes the input field required and displays an asterisk next to the label.
showDuration Boolean false This is used with the Date Range Time Picker component. When true, the duration of the time is displayed. The default is false.
size String sm, lg This changes the size of the input field. The default is default.
validation Boolean false When true, this will display the input in red.
validationMessage String This is the message that will be displayed when the input field is in a validation state.
value String This is the value of the input field.
warningMessage String This is the message that will be displayed when the input field is in a warning state.
