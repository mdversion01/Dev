Using Component in other JavaScript Frameworks

Integrating a Lit Element component with other JavaScript frameworks like Vue, React, and Angular involves creating a wrapper component in each framework to encapsulate and use the Lit Element component. Here are examples for each framework:

Vue.js

Install the necessary packages:

Make sure you have the necessary dependencies installed. You might need to install @webcomponents/webcomponentsjs for polyfills if you need to support older browsers. 

        
          npm install @webcomponents/webcomponentsjs

Import the LitElement component in your Vue component:

          
            <template>
            <div id="app">
            <timepicker-manager
                  :ariaLabel="'Time Picker'"
                  :ariaLabelledby="'time-label'"
                  :id="'time-input'"
                  :inputName="'time'"
                  :is24HourFormat="true"
                  :size="''"
                  :validationMessage="''"
                  :display24HourOnly="false"
                  :display12HourOnly="false"
                  :hideTimepickerBtn="false"
                  :isValid="true"
                  :hideSeconds="false"
                  :usePlTimePicker="false"
                  @change="handleTimeChange"
                  ></timepicker-manager>

                  <p>Selected Time: {{ selectedTime }}</p>

                </div>
                </template>
            
                <script>
                  export default {
                    name: "App",
                    data() {
                      return {
                        selectedTime: '',
                      };
                    },
                    methods: {
                      handleTimeChange(event) {
                        this.selectedTime = event.detail.value;
                      },
                    },
                  };
                </script>

Include polyfills in your main.js or index.js:

          
            import '@webcomponents/webcomponentsjs/webcomponents-bundle.js';
            import { createApp } from 'vue';
            import App from './App.vue';

            createApp(App).mount('#app');
React

Install the necessary packages:

        
          // bash
          npm install @webcomponents/webcomponentsjs

Import the LitElement component in your React component:

          
            import React from 'react';
            import ReactDOM from 'react-dom';
            import 'path/to/timepicker-manager.js';
            import '@webcomponents/webcomponentsjs/webcomponents-bundle.js';

            function App() {
              const [selectedTime, setSelectedTime] = useState('');
            
              useEffect(() => {
                const timepickerManager = document.querySelector('timepicker-manager');
                const handleTimeChange = (event) => {
                  setSelectedTime(event.detail.value);
                };
            
                timepickerManager.addEventListener('change', handleTimeChange);
            
                return () => {
                  timepickerManager.removeEventListener('change', handleTimeChange);
                };
              }, []);
              return (
                <div className="App">
                <timepicker-manager
                    ariaLabel="Time Picker"
                    ariaLabelledby="time-label"
                    id="time-input"
                    inputName="time"
                    is24HourFormat={true}
                    size=""
                    validationMessage=""
                    display24HourOnly={false}
                    display12HourOnly={false}
                    hideTimepickerBtn={false}
                    isValid={true}
                    hideSeconds={false}
                    usePlTimePicker={false}
                    ></timepicker-manager>
                  <p>Selected Time: {selectedTime}</p>
                  </div>
              );
            }

            ReactDOM.render(, document.getElementById('root'));
Angular

Install the necessary packages:

        
          // bash
          npm install @webcomponents/webcomponentsjs

Add the LitElement component to your angular.json:

        
          "scripts": [
            "node_modules/@webcomponents/webcomponentsjs/webcomponents-bundle.js"
          ]

Import the LitElement component in your Angular component:

          
            // src/app/app.component.ts
            import { Component, OnInit } from '@angular/core';
            import 'path/to/timepicker-manager.js';
            
            @Component({
              selector: 'app-root',
              template: `
              <timepicker-manager
                [ariaLabel]="'Time Picker'"
                [ariaLabelledby]="'time-label'"
                [id]="'time-input'"
                [inputName]="'time'"
                [is24HourFormat]="true"
                [size]="''"
                [validationMessage]="''"
                [display24HourOnly]="false"
                [display12HourOnly]="false"
                [hideTimepickerBtn]="false"
                [isValid]="true"
                [hideSeconds]="false"
                [usePlTimePicker]="false"
                (change)="handleTimeChange($event)"
                  ></timepicker-manager>

                  <p>Selected Time: {{ selectedTime }}</p>
              `,
              styleUrls: ['./app.component.css']
            })
            export class AppComponent implements OnInit {
              selectedTime = '';
            
              ngOnInit() {}
            
              handleTimeChange(event: any) {
                this.selectedTime = event.detail.value;
              }
            }

Include polyfills in your polyfills.ts:

        
          import '@webcomponents/webcomponentsjs/webcomponents-bundle.js';

Summary

For each framework, you need to:

  1. Listen for custom events (like change) emitted by the LitElement component.
  2. Update the framework's state with the new value from the event.

The handleTimeChange function will capture the emitted value from the TimePicker or PlTimePicker component and store it in a reactive state (data property in Vue, state in React, or class property in Angular).

Adjust the paths and imports based on your project structure. The provided examples showcase how to create wrapper components for each framework to use your Lit Element 'Table' component.

The interoperability is possible because Web Components, once defined and registered as custom elements, act like native HTML elements that these frameworks can understand and interact with.
